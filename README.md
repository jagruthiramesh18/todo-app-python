
A simple command-line To-Do List application built using Python.

## Features
- Add tasks
- View tasks
- Delete tasks
- Save tasks to file
- Persistent storage using text file

## Technologies Used
- Python
- File Handling
- Functions
- Lists

##  How to Run

1. Clone the repository
2. Run the Python file

python Todolist.py

##  Project Structure

todo-app-python/
│── Todolist.py
│── README.md
│── tasks.txt (auto-created)
